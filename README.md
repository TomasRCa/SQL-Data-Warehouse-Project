# SQL-Data-Warehouse-Project
#### For this project, I'm designing and building a Data warehouse, aiming to showcase SQL, DataWarehouse Architecture and ETL skills as a portfolio project.
---
## **📜This project includes:**
##### Data Architecture: Designing a Data Warehouse using a Medallion Architecture: Bronze, Silver, and Gold layers.
##### ETL Pipelines: Extracting, transforming, and loading data from source system into the warehouse.
##### Data Modeling: Developing fact and dimension tables tailored for analytical queries.
##### Analytics & Reporting: Creating SQL-based reports and dashboards to extract insights.
